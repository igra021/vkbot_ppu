🤖 VK Chat Bot — консультант по утеплению домов
ВК-бот для автоматизации консультаций по утеплению домов напылением ППУ. Использует OpenAI для генерации ответов и RAG для поиска по базе знаний.

🚀 Возможности
✅ Интеллектуальное общение — использует OpenAI GPT для генерации ответов

✅ Хранение истории диалогов — каждый пользователь имеет свою историю в SQLite

✅ RAG-поиск — поиск по базе знаний (вопросы/ответы экспертов)

✅ Администрирование — команды для управления ботом (только для админа)

✅ Обработка вложений — поддерживает фото, видео

✅ Пересылка медиа — автоматическая пересылка фото/видео администратору

✅ Гибкая конфигурация — настройки через .env файл

✅ Логирование — подробные логи с loguru

🛠 Стек технологий
Компонент	Технология
Фреймворк для VK API	vkbottle
LLM	OpenAI API (Chat Completions)
Язык	Python 3.10+
База данных	SQLite (aiosqlite)
Эмбеддинги	sentence-transformers
Логирование	loguru
Управление переменными	python-dotenv
⚙️ Установка и запуск
Клонирование репозитория
bash
git clone https://github.com/igra021/vk-bot.git
cd vk-bot

🔧 Конфигурация
Переменные окружения
Переменная	Описание	Обязательно
VK_TOKEN	Токен доступа к VK API	✅
ADMIN_ID	VK ID администратора	✅
OPENAI_API_KEY	API ключ OpenAI	✅
OPENAI_MODEL	Модель OpenAI (по умолчанию gpt-4o-mini)	❌
DB_PATH	Путь к файлу БД (по умолчанию conversations.db)	❌
DEBUG	Режим отладки (True/False)	❌

📌 Команды бота
Команда	Описание	Доступ
/halt	Остановка бота	Только админ
/start	Начало диалога	Все пользователи
/reset	Сброс истории диалога	Все пользователи
/help	Справка	Все пользователи

🔨 Что сделать
Запустить RAG — интегрировать поиск по базе знаний.

Доделать пересылку вложений админу — реализовать автоматическую отправку фото/видео от пользователей администратору.

Админ даёт оценку вложениям — сообщения админа передавать в LLM как assistant. Продумать механизм реакции бота на сообщения администратора.

Формат базы RAG — решить, использовать ли Excel (.xlsx) вместо .csv для базы знаний.

Перенос базы знаний из промпта в RAG — разделить данные на категории: общая база знаний, цены, преимущества, технические характеристики и т.д.
